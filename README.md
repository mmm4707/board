## 기본 기능  
-사용자는 게시물을 조회하거나 생성, 수정, 삭제 가능<br>
-Jpa의 pageable을 이용해서 페이징 구현<br>
-Spring Data JPA를 사용해 데이터베이스와 상호작용하는 코드를 작성<br>
-@Modifying, @Query 어노테이션을 사용하여 특정 쿼리를 정의<br>
